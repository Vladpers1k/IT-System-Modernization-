Список основних завдань (тасок) для ІТ-відділу ПАТ «Вінницький асфальтобетонний завод»

1. Налаштування серверного середовища

Встановлення операційної системи (Windows Server / Linux)

Створення облікових записів користувачів

Налаштування резервного копіювання

2. Розгортання бази даних підприємства

Вибір СУБД (MySQL / PostgreSQL)

Створення таблиць: постачання, виробництво, персонал, техніка

Тестування підключення клієнтів до БД

3. Інтеграція з виробничими процесами

Налаштування обліку сировини (цемент, щебінь, бітум)

Впровадження системи контролю якості продукції

4. Розробка веб-інтерфейсу для персоналу

Реєстрація користувачів

Перегляд даних із бази

Панель адміністратора

5. Тестування інформаційної системи

Перевірка коректності введення / збереження даних

Перевірка швидкодії

Звіт про виявлені помилки

6. Впровадження в експлуатацію

Перенесення системи на сервер підприємства

Проведення навчання користувачів

Підготовка технічної документації
